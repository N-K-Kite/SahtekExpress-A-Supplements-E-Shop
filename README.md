# SahtekExpress-A-Supplements-E-Shop
A full-stack e-commerce web application for sports supplements developed using Spring Boot and Spring Security. The project follows a clean layered architecture (Controller, Service, Repository) with DTO-based data exchange and JWT authentication.


 Technical Stack :

Backend: Spring Boot, Spring Security, Maven

Authentication: JWT (stateless authentication)

Database: MySQL / PostgreSQL (JPA / Hibernate)

API: RESTful services with DTO ↔ Entity mapping

Frontend: JSON-based client consuming REST APIs

Architecture: Controller → Service → Repository

Authentication & User Management :

User registration and login with JWT-based authentication

Secure role-based access (USER / ADMIN)

Profile management

User account moderation (admin block/unblock)

Client Features (Storefront) :

Product catalog with categories

Product search and filtering

Product detail pages

Fully functional shopping experience

 Cart & Order Processing :

Add/remove products from cart

Cart validation and checkout flow

Address selection

Order creation and persistence

 Payment & Delivery (Simulation)

Payment simulation workflow

Delivery process simulation (order lifecycle)

 Admin Dashboard :

Product & Category Management (CRUD)

User management

Order validation
